# SkillProof ATS

SkillProof ATS is an evidence-driven Applicant Tracking System that evaluates
candidates based on demonstrated skills, semantic matching, and growth potential
instead of keyword-based resume filtering.

## Problem
Traditional ATS systems rely heavily on keyword matching and resume formatting,
often rejecting skilled candidates without understanding real capability.

## Solution
SkillProof ATS combines semantic AI matching, portfolio proof analysis, and
micro-skill evaluation to rank candidates more fairly and accurately.

## Tech Stack
- Backend: FastAPI (Python)
- ML/NLP: Sentence Transformers, scikit-learn
- Frontend: React
- Database: PostgreSQL
- Vector Search: FAISS
